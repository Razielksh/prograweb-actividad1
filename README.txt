Actividad 1.- Ejercicios html css js boostrap

Descripción de la actividad
Este repositorio integra los ejercicios prácticos desarrollados durante la Actividad 1 de la materia de 
Programación Web.

Cada ejercicio aborda distintos conceptos fundamentales del desarrollo web: estructura HTML, estilos con CSS 
y la implemetación de código con JavaScript.

Contenido:
Ejercicio 1 al 6: Ejercicios de práctica sobre HTML, manejando: párrafos, cabeceras, imagenes, etc.
Ejercicios 7 al 9: Implementación de CSS en el documento HTML
Ejercicio 10: Calculadora interactiva que convierte grados Celsius a Fahrenheit, usando la fórmula F = (C × 9/5) + 32. 
Incluye validación de campos (vacío y numérico) y un campo de resultado de solo lectura (readonly). Impelmentación de
HTML + CSS + JavaScript.
